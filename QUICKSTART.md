# 快速部署指南 🚀

## 立即啟動 (3 步驟)

### 1️⃣ 設定 API 金鑰

```bash
copy .env.example .env
```

編輯 `.env` 填入：

```
VITE_GEMINI_API_KEY=你的金鑰
```

👉 取得金鑰：<https://makersuite.google.com/app/apikey>

### 2️⃣ 啟動系統

```bash
npm run dev
```

### 3️⃣ 開啟瀏覽器

```
http://localhost:5173
```
